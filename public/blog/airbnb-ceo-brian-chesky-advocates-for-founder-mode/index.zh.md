# Airbnb CEO Brian Chesky的“创始人模式”哲学

![Airbnb CEO Brian Chesky Advocates for "Founder Mode" Leadership](/images/blog/airbnb-ceo-brian-chesky-advocates-for-founder-mode.png)

在最近的一次采访中，Airbnb CEO Brian Chesky分享了他的“创始人模式”哲学。这种管理理念主张创始人在公司发展中应保持对产品、用户和公司文化的紧密关注，而不是简单地采用传统的“管理者模式”。

Chesky指出，传统的“管理者模式”的领导方式——雇佣优秀的人才、授权管理、通过层级进行控制并避免涉及细节——导致了Airbnb在扩张过程中失去连贯性。他感到公司在2019年时变得支离破碎、官僚化且方向不明。在新冠危机的推动下，他开始重新掌控，深入参与产品和运营，以创始人领导原则重建Airbnb。

## 创始人模式的核心要素

根据Chesky的观点，创始人应当坚持以下几个方面：

1. **简约与极致细节**：Chesky受Apple的Hiroki Asai的启发，主张简约并不是减少，而是将事物精炼到其核心。他认为，优秀的公司会注重细节，因为完美的输入最终会导致强大的输出。

2. **从紧到松的管理**：创始人应当在初期与团队密切合作，教导标准并建立“肌肉记忆”，在此基础上再进行授权。如果没有共享标准的前提进行授权，只会导致混乱。

3. **小型精英团队胜过大型官僚机构**：Airbnb的“Project Hawaii”项目展示了将优秀人才集中在一个狭窄但高杠杆问题上的巨大影响力。通过专注于从搜索到预订的转化，该项目据称带来了数亿美元的额外收入。

4. **AI对管理者的威胁**：“AI不仅威胁员工，也威胁管理者”。Chesky强调，只有管理人员的领导者可能会失去相关性，而真正理解产品、用户、文化和判断的领导者会变得更有价值。

5. **创始人模式与微观管理的区别**：虽然看似相似，但“创始人模式”强调的是教导口味、对齐判断、去除模糊性与构建组织能力。

6. **输入胜过结果**：Chesky提倡专注于改善输入，如产品质量、用户体验、团队能力和操作细节，而非单纯追求业绩指标，如收入和市场份额。

最终，Chesky认为，Airbnb的长期价值来自于其品牌、社区、身份认同、信任和现实世界的联系。这些因素远比软件更为持久。他坚持创始人在招聘前200个核心员工时应亲自参与，因为每一位优秀的员工都能降低未来的管理负担。
